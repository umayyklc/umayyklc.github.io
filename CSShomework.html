CSS Temel Ödevi

Verilen benzer bir düzenle basit bir web sayfası oluşturun. CSS'in temel özelliklerini uygulayın. Sadece HTML5 + CSS3 kullanın.

İçerik

Bu alanda örnek içerik metinleri bulunmaktadır. CSS özellikleriyle bu metinler arasında boşluklar (line-height) ve hizalamalar düzenleriz.

Önce CSS ile kutuları yan yana getirmeyi öğrendik. Şimdi ise CSS seçicilerini kullanarak grup ve hiyerarşik seçimler yapacağız.

Yan Menü / Bilgi

Bu alan yan menü veya bilgi kısmıdır. CSS ile genişlik (width), yükseklik (height) ve kenar boşluklarını kullanarak düzenleyebiliriz.

Öğrendiklerimizi hiyerarşik ve grup seçicilerle uygulayıp farklı bölümlere stil verebiliriz.
